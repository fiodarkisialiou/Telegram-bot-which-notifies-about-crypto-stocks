Personal Stock & Crypto Alert System
A Python script that runs in the background and monitors stock and cryptocurrency prices in real time. When a price crosses a threshold you define — either too high or too low — it instantly sends you a message on Telegram. Every morning at 9:00 AM it also sends a daily digest with the current price of everything in your watchlist.
Built with yfinance for stock data, CoinGecko API for crypto, schedule for timing, and Telegram Bot API for notifications.
